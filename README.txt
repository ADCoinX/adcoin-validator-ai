
# ADCoin Validator (LIVE)

## Features:
- Live API validation for Ethereum wallets (balance + transactions)
- TRON and BTC structure included, real API integration coming
- Mobile-friendly crypto-style UI
- Scam-safe validator interface

## Deploy on Render:
1. Go to https://render.com
2. Create new Web Service
3. Connect to GitHub or upload manually
4. Use:
   - Start command: python app.py
   - Runtime: Python 3
   - Include requirements.txt
